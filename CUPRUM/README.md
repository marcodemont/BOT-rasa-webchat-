# CUPRUM (leer / Werkstatt-Template)

Aktueller Stand:

- **AURUM ist jetzt die Haupt-App** (inhaltlich ex-CUPRUM).
- Dieser `CUPRUM/`-Ordner bleibt bewusst **leer als Werkstatt-Template**,
  bis neue Dateien sauber einsortiert hochgeladen werden.

## Geplante Struktur

```text
CUPRUM/
  src/
  public/
  supabase/
```

## Hinweis

Wenn du neue Werkstatt-Dateien hochlädst, bitte direkt in diese Struktur legen
(nicht ins Projekt-Root), dann kann `CUPRUM.bat` wieder als eigener Starter dienen.
